# imgToBraille
A small python script to turn a image into a terminal friendly set of Unicode braille characters. 
Uses a function found from an anonymous post  to turn a grid into braille output. 
This program aims to draw the boarder between colours instead of using a light threshold and thus should give a clearer 
img especialy for larger images.

Works far better on darker backgrounds.
